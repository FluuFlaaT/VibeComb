### A All-in-One configuration manager for AI coding tools

## Concepts

- **Provider**: A service that provides AI coding tools.
- **MCP**: A middleware that coordinates the interaction between providers and users.
- **Skill**: A function that can be performed by a provider.
- **Prompt**: A template that guides the user in the interaction with a skill.
- **Profile**: A collection of skills and prompts that can be used to generate a configuration.
